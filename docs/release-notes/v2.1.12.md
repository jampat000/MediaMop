# MediaMop v2.1.12

This patch release polishes sidebar branding layout and refreshes updater regression coverage used for release validation.

## Highlights

- Refined sidebar brand presentation:
  - cleaned up duplicate title rendering in the shell header
  - switched sidebar/auth logo rendering to the cropped logo asset for better visual balance
  - tuned logo sizing and vertical spacing so tagline, divider, and section labels read cleanly
- Stabilized Windows updater regression tests around stale-state reconciliation paths to prevent false failures during release gating.

## Support notes

- No billing, licence, or feature-gating behavior was added.
- In-app upgrade verification and trusted installer validation behavior are unchanged from previous updater fixes.
