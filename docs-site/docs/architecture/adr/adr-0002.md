# ADR-0002: Database and Alembic (SQLite-first)

**Status:** Accepted

MediaMop uses file-backed SQLite as its primary database with Alembic for schema migrations. No external database service is required.

> Full ADR source: [`docs/adr/ADR-0002-database-sqlite-alembic.md`](https://github.com/jampat000/MediaMop/blob/main/docs/adr/ADR-0002-database-sqlite-alembic.md)
