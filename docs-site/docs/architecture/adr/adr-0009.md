# ADR-0009: Suite-wide Timing Isolation for Durable Work

**Status:** Accepted

Operator-controlled intervals, schedules, cooldowns, retries, last-run, and timing-based pruning follow a per-module, then per-job-family isolation model for durable work coordination.
